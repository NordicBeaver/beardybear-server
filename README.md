## Description

Server for the BeardyBear project. Built with [Nest](https://github.com/nestjs/nest).

Live version: https://beardybear.nordicbeaver.io/

For more details: https://www.youtube.com/watch?v=b_QDY1UIT7Y&list=PL7b0cPjh8z6K_4qLNu_QdE3RpiU6omViC

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
